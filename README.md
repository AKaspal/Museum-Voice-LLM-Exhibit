## üßë‚Äçüíª About

**Museum AI Scientist Interaction** is an interactive platform that leverages Flask and React to simulate conversations with renowned historical scientists such as Isaac Newton, Marie Curie, Galileo Galilei, Dmitri Mendeleev, and Albert Einstein. Users can engage in spoken dialogues with these AI-driven personalities, receive transcriptions of their speech, and hear synthesized audio responses, creating an immersive educational experience.

## üöÄ Technologies Used

### **Backend**
- [Flask](https://flask.palletsprojects.com/) - Web framework for Python
- [Flask-CORS](https://flask-cors.readthedocs.io/) - Handling Cross-Origin Resource Sharing
- [pyttsx3](https://pyttsx3.readthedocs.io/) - Text-to-speech conversion
- [Vosk](https://alphacephei.com/vosk/) - Speech recognition toolkit
- [Python-dotenv](https://github.com/theskumar/python-dotenv) - Managing environment variables
- [Requests](https://docs.python-requests.org/) - HTTP library for Python

### **Frontend**
- [React](https://reactjs.org/) - JavaScript library for building user interfaces
- [Axios](https://axios-http.com/) - Promise-based HTTP client
- [react-icons](https://react-icons.github.io/react-icons/) - Icon library for React

### **APIs**
- [Arli.ai](https://arli.ai/) - AI-generated responses

## ‚ú® Features
- **Historical Scientist Personalities**: Interact with AI models of famous scientists, each with unique conversational styles.
- **Speech Recognition and Text-to-Speech**: Speak to the AI and receive both textual and audio responses.
- **Multi-language Support**: Choose from multiple languages for AI responses.
- **Conversation History**: Maintain a history of interactions for context-aware dialogues.
